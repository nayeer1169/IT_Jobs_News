## Skills In Demand

Prioritise skills that employers can test quickly:

- Java or Python
- SQL and database basics
- JavaScript and React
- REST APIs
- Git and GitHub
- Linux basics
- cloud basics: AWS, Azure, or GCP fundamentals
- QA automation
- data analytics
- GenAI tool usage and prompt/workflow automation

## Fresher Action Plan For Today

1. Apply through official portals first: TCS, Infosys, Accenture, IBM, Wipro, HCLTech, and Cognizant.
2. Keep 2-3 proof projects ready: one deployed full-stack app, one SQL/API project, and one automation/testing project.
3. Prepare aptitude, DSA basics, OOP, DBMS, OS/networking, and resume-based questions.
4. Apply beyond mass hiring: startups, GCCs, apprenticeships, internships, support engineer roles, QA roles, junior web developer roles, and customer-facing technical roles.
5. Track applications in a spreadsheet with company, role, source link, date applied, status, and follow-up date.

## Reliability Notes

- Official company career pages are safer than third-party listings.
- Fresher hiring intent surveys indicate market direction, not guaranteed vacancies at every company.
- Job openings change daily; verify the role is live before applying.
